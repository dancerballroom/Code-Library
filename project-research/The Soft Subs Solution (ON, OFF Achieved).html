Pseudo-Code

1.  Chop off Ends of Movie.mp4
2.  Separate Movie.mp4 from its Subtitles.srt (Commercial Removal HERE)
3.  X = Number of Milliseconds to add to Length of Chapter
4.  Add a line at the bottom of the SRT

FORMAT of the last 3 lines of each CHAPTER#.SRT
Line 1:
(Last Line Number) + 1

Line 2:
(HH:MM:SS,fff --> HH:MM:SS,fff)
// where
//     HH = The Hours Length of the Chapter
//     MM = The Minutes Length of the Chapter
//     SS = The Seconds Length of the Chapter
//     fff = X (The Milliseconds Length of the Chapter)

Line 3:
<font face="Serif"><font size="18"></font></font>
// TRICK:
// There is no Text.  Just a Place-Holder.
// (Without any words between the <font></font> tags, nothing shows.)

// Why?  This last line establishes the Exact Lenth of the File
//       when using "Subtitle Edit" to Join the Chapters Exactly.

5. Join the SRT Subtitles in "Subtitle Edit".  Concatenate the resulting container files 
   listed in the text file.
   
Edit with "Subtitle Editor" / Tools / Join SRT Subtitles
# REPLACES FAIL:  for %a in (Chapter*.srt) do echo file '%a' >> srtfiles.txt
#           and:  ffmpeg -f concat -safe 0 -i srtfiles.txt -c copy everything.srt
