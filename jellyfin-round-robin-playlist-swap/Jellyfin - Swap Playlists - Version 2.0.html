Jellyfin - Swap Playlists - Version 2.0

Purpose:  The Jellyfin Movie Server has very poor playlist management.  This program allows on the fly swapping of playlists with a single Double-Click of a Desktop Shortcut, and a Jellyfin Library Refresh.

  • The Directory Location of Jellyfin Playlist “Classic Series - SWAP”

  • The SWAP Directory pre-loaded with Alternative Playlists.











  •  Clicking Swap.bat and Refreshing the Jellyfin Playlist (BEFORE)
  • Clicking Swap.bat and Refreshing the Jellyfin Playlist (AFTER)
  • Clicking Swap.bat and Refreshing the Jellyfin Playlist (Another Time)

Pre-requisites

  • Jellyfin Installed on Windows machine
  • At least One Created Playlist
    • File Manager Location (with Hidden Files turned on)
      • C:\ProgramData\Jellyfin\Server\data\playlists\[New Playlist Directory]
  • One Subdirectory “SWAP”
    • C:\ProgramData\Jellyfin\Server\data\playlists\[New Playlist Directory]\SWAP
  • Two Files in the [New Playlist Directory]:  
    • “Add_Playlist.bat”      # add playlists 
    • “SWAP.bat”                # SWAP playlists after there are playlists to SWAP.

Recommended Setup:

After Creating the first playlist, locating the playlist, placing Add_Playlist.bat and SWAP.bat in the Playlist directory, and Creating the Subdirectory SWAP.
  • Create a New Playlist using the Jellyfin Movie Server GUI.
  • Locate the “playlist.xml“ for the New playlist.
  • Drag/Drop the “playlist.xml” Created by the Jellyfin GUI onto Add_Playlist.bat
  • Et Voila!  Delete the Second Playlist you just created using Jellyfin
  • Run SWAP.bat (to test the SWAP function)
  • Refresh Playlist in the Jellyfin GUI
  • Add other playlists and the SWAP LIST will grow.

1 of 2 - Add_Playlist.bat

@Echo Off
setlocal enabledelayedexpansion

REM # Change this to the SWAP directory below the current Playlist Directory
pushd "C:\ProgramData\Jellyfin\Server\data\playlists\Classic Series - SWAP\SWAP"

REM # Count Backup Files
for /f %%A in ('dir *Playlist.* ^| find "File(s)"') do set cnt=%%A

REM # Calculate Next Number following existing playlists
SET /A cnt+=1

REM # Add Leading Zeroes to the filename
if %cnt% LEQ 9 SET Zeroes=00
if %cnt% GEQ 10 SET Zeroes=0
if %cnt% GEQ 100 SET Zeroes=

REM # Make a copy of the filelist dropped onto "Add_Playlist.bat"
copy %1 "!Zeroes!%cnt% - Playlist.xml"

popd

2 of 2 - SWAP.bat

@Echo Off
setlocal enabledelayedexpansion

REM # Create New File From %1 variable
SET SWAP_DIR=SWAP

REM # Copy First-File to SWAP directory
MOVE "Playlist.xml" "%SWAP_DIR%\Playlist.xml"

REM # Make the SWAP Directory the Current Directory
pushd SWAP

REM # Count Backup Files
for /f %%A in ('dir *Playlist.* ^| find "File(s)"') do set cnt=%%A

REM # Calculate Zeroes Padding for Filenames
if (%cnt% LEQ 9) ( SET Zeroes=00)
if (%cnt% GEQ 10) ( SET Zeroes=0)
if (%cnt% GEQ 100) ( SET Zeroes=)

REM # Exit the SWAP directory
popd

REm # Copy SWAP\001_playlist.xml to playlist.xml
MOVE "%SWAP_DIR%\%Zeroes%1 - Playlist.xml" Playlist.xml

pushd SWAP

REM # RE-Count Files
for /f %%A in ('dir *Playlist.* ^| find "File(s)"') do set cnt=%%A

REM # Rename other Backup playlist.xml files
for /l %%x in (2, 1, %cnt%) do (

    set /a cnt3+=1
    if !cnt3! LEQ 9 SET Zeroes=00
    if !cnt3! GEQ 10 SET Zeroes=0
    if !cnt3! GEQ 100 SET Zeroes=

    if %%x LEQ 9 SET Zeroes1=00
    if %%x GEQ 10 SET Zeroes1=0
    if %%x GEQ 100 SET Zeroes1=

    MOVE "!Zeroes1!%%x - Playlist.xml" "!Zeroes!!cnt3! - Playlist.xml"
)

REM # Zero Placeholders Again
set /a cnt3+=1
if !cnt3! LEQ 9 SET Zeroes=00
if !cnt3! GEQ 10 SET Zeroes=0
if !cnt3! GEQ 100 SET Zeroes=

if %%x LEQ 9 SET Zeroes1=00
if %%x GEQ 10 SET Zeroes1=0
if %%x GEQ 100 SET Zeroes1=

REM # Give the first Playlist a Number
MOVE "Playlist.xml" "%Zeroes%%cnt3% - Playlist.xml"

REM # FOR DEBUG ONLY
Echo Current : 
Type ..\Playlist.xml
Echo Archived : 
REM # View Contents of each File
for /l %%x in (1, 1, %cnt%) do (
    if %%x LEQ 9 SET Zeroes=00
    if %%x GEQ 10 SET Zeroes=0
    if %%x GEQ 100 SET Zeroes=

    REM # Show Filename
    Type "!Zeroes!%%x - Playlist.xml"
)
popd
REM # UNREM Next Line to DEBUG
REM pause
REM # END DEBUG
