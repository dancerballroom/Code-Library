Windows MkLink Batchfiles (Virtual Directories)

Setting up a Virtual Path for Snap2HTML and Playlist Generator

Purpose:  Limit Search Path to Specific Directories used by the Jellyfin Movie Server.
Method:  Use Windows MkLink to create hard links to the External Hard Drive Directories. 
      
     1.   The following is my Virtual Directory HOME  
C:\>tree JELLYFIN_HOME
Folder PATH listing
Volume serial number is AA96-C106
C:\JELLYFIN_HOME
├───HDD-1
├───HDD-2
├───HDD-3
├───HDD-4
└───HDD-5
  • Each batchfile goes in the corresponding Virtual subdirectory.
  • The Hard Links correspond to Subdirectories existing on External Hard Drives.
