01-DIR-TREE-GEN.bat - Dynamic Web Site builder

Purpose:  

  • Save Backup of QUIP documents on Google DRIVE.
  • Creates the Folder Structure on the HOST computer.
  • Creates the HTML using FRAMES to generate Navigation.

Preparation:

  • Download your QUIP Files into your desired Tree Structure.  See folder-TREE
    • I recommend to download to a TEST-DATA directory with the same desired TREE as on the FINAL-DATA directory.
    • You can copy into the original QUIP HTML FILES to a PRODUCTION-SERVER Directory later
  • Create a New Unique Google G-mail E-mail.  Find the link to Google Drive and set it up for use.
    • This Google DRIVE will only host your backup QUIP HTML Files and Directories.
  • After the Files and Folders are uploaded to Google DRIVE
    • Set all file permissions to public
    • Set all folder permissions to public.
    • Double check this for the current directory(Folder) and all subdirectories(Sub-Folders)
  •  Visit drv.tw and set up your Google DRIVE according to the instructions.
    • Example:  https://larrys-today-in-coding.on.drv.tw/

NOTE:  Common Errors:  

  • Not setting Folders and Files to Public on Google DRIVE.
  • The Browser doesn’t refresh without a FORCED refresh if changes have been made.
    • Control+Shift+R

Later Addition: 

Use free Windows software: SNAP2HTML to Generate a Search component.
Edit the Generated HTML (I use NotePad++)
  • Delete any local Folder Path references on the SNAP2HTML Generated Webpage.

Free Web Page Storage Option:

  • After Web pages generated and fully tested and working, mount to Google Drive
  • For Updates:
    • Save new QUIP pages to Windows Data-Tree
    • For Generating main.html Directory containing HTML Documents:  HTML_FILES_GEN-main.bat
    • For Generating main.html Directory containing Sub-Directories: HTML_DIRS_GEN-main.bat
    • Generate and Test the local HTML (click index.html on the root) 
    • Upload to Google DRIVE.
    • Refresh link at  drv.tw 
    • Force Refresh the Browser (I use Chrome) after changes using Control+Shift+R

Steps

  • How do you want your Folders and Sub-Folders to be?
  • What html files do you want in your folders?
  • Design the TREE-GEN-TEST-DATA to create the Directory Structure.
  • Run DIR-TREE-GEN.bat
  • Copy the TREE-GEN-TEST-DATA into the newly created Directory Tree.
  • Follow directions and Run HTML_FILES_GEN-main.bat in each File directory
  • Mount to Google DRIVE
  • Set Permissions to Public (All Files and Directories)
  • Set up DRV.TW (See Free Webpage Storage Option)
  • Share your link!
  • All Done!

NOTE:  This program only allows Folders to contain Just Folders or Just Files.  More Later...

DIR-TREE-GEN.bat - also - 01-DIR-TREE-GEN.bat

@echo off
Setlocal enabledelayedexpansion
set file_resources=%~d0%~p0resources\FILES\*.*
set dir_resources=%~d0%~p0resources\DIRS\*.*
set dir_resources2=%~d0%~p0resources\DIRS\resources
set root_index=%~d0%~p0resources\root_index.html
set root_menu=%~d0%~p0resources\root_menu.html
set search_html=%~d0%~p0resources\Search.html
set folder_icon=%~d0%~p0resources\Folder.jpg
set document_icon=%~d0%~p0resources\Menu.jpg
set main_html=main.html
set /a DIRLevel=1
set CRLF=^


:top
::Main Menu
set root_prompt="0 = Exit!CRLF!1 = Create Directory!CRLF!2 = Create File!CRLF!3 = To Parent or TREE!CRLF!: "

set /p menuitem=!root_prompt!
REM :: Menu Choices
REM :: 0 = Exit Program
IF "!menuitem!"=="0" (
    Set /a DIRLevel = 0
    copy ^"!root_index!^" index.html
    copy ^"!root_menu!^" menu.html
    copy ^"!folder_icon!^" .
    copy ^"!document_icon!^" .
    copy ^"!search_html!^" .
    
    Echo Level: !DIRLevel!
    tree
    echo Goodbye!
    pause
    Exit 0
) ELSE IF "!menuitem!"=="1" (
    set /a DIRLevel+=1
    Echo Level: !DIRLevel!
    set DIR_prompt="DIR-NAME -> "
    set /p DIR-Name=!DIR_prompt!
    md "!DIR-Name!"
    echo run directory script > run-dir.txt
    pushd "!DIR-Name!"
    echo run files script > run-file.txt
    copy "!dir_resources!" .
    md resources
    xcopy "!dir_resources2!" .\resources /r /d /h /y
    goto :top
) ELSE IF "!menuitem!"=="2" (
    REM set FILE_prompt="FILE-NAME-> "
    REM set /p File-Name=!FILE_prompt!
    copy "!file_resources!" .
    REM copy /b NUL "!FILE-Name!"
    goto :top
) ELSE IF "!menuitem!"=="3" (
    set /a DIRLevel-=1
    Echo Level: !DIRLevel!
    popd
    tree
    goto :top
) ELSE (
  echo Try Again
  goto :top
)
