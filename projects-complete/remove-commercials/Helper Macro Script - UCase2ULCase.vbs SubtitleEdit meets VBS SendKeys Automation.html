UCase2ULCase.vbs

' UCase2ULCase.vbs
' which in short means 
' Upper Case to Upper Case and Lower Case Subtitle Conversion 
' ScriptEngine
' Visual Basic Script for Windows

' #To Use:
' #1. Open a .srt Subtitle File to check if all upper case
' #   if yes, then close.
' #2. Double click this file and it converts to normal uppercase/lowercase.

' #Create a Shell Object
Set WshShell = WScript.CreateObject("WScript.Shell")
' #Open Subtitle Edit
WshShell.Run """C:\Program Files\Subtitle Edit\SubtitleEdit.exe""", 9

' #Give Subtitle Edit time to load
WScript.Sleep 500 

' #Control + Shift + C   
' #Next:  SubtitleEdit Shortcut => Change Casing Screen
WshShell.SendKeys("^+C")    

WScript.Sleep 500    ' #Command Load Time

WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{ENTER}"

WScript.Sleep 1000    ' #Command Load Time

' #Next: Subtitle Edit Casing Verification Screen

WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{TAB}"
WshShell.SendKeys "{ENTER}"

WScript.Sleep 1000  ' Command Load Time

' #Next: Exit Subtitle Edit

WshShell.SendKeys "%{F4}"

WScript.Sleep 500  

' #Next: Exit Verification Save: Yes, Save
WshShell.SendKeys "{ENTER}"

Screen Shots of the Accomplishments of the Pseudo Macro/Programmlette’s Accomplishment’s

Without the Macro File you have these Motions or Steps:

  • Open File and Verify all Upper Case
  • “Change Casing Shortcut”  Control + Alt + C
  • Click OK
  • “Verify Casing“ Pop-Up 
  • Click OK
  • “Click X” Close File
  • “Save File” Pop-Up Click OK

With the Macro File you have 3 Motions

  • Open File and Verify all Upper Case
  • Close the File, Done if OK
  • If Not OK, Run Macro
