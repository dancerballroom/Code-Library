Add Function to Generated Snap2HTML Online - JellyFin Home Listing - Redirect to IMDB

Problem  Generated Movie and Series have no descriptions or Photos associated with them.

Solution Run Snap2HTML and change one line of a function buried in the HTML to create a link to the IMDB (Internet Movie Database)

Picture of Result 

  • 1st Picture: Movie and Series Listing 
  • 2nd Picture: IMDB Search
  • 3rd Picture: Movie Information

Pre-Requisite:  The Snap2HTML has already run to create the linked HTML

The Replacement Code:  COPY/PASTE Version

(Somewhere around Line 6490, depending upon the Generated Script)

// BEFORE: The Link is to a Local File
// Larry: CODE to REPLACE
file_tmp = linkProtocol + linkRoot + dir_tmp + sTmp[0] + "\">";
                            
// AFTER: The link is to IMDB to do a search on the Movie or Series
// Larry:  NEW CODE (Copy the Next Line)
file_tmp = "https://www.imdb.com/find/?q=" + sTmp[0] + "\" target=\"_new\">";

The Replacement Code: The How to Find it Version

HINT:  Using the NotePad++ Editor Look for:   onlyLinkExtensions.length
(About 19 to 20 PGDN Hits after the last DATA line.)
NOTE Reminder:  target = “_new“ just opens the link in a new browser window

Et Voila!  Woopie! (pronounced “Voopie!” auf Deutsch)
