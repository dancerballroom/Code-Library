5-Drive-Replace

Purpose:  MKLink Directories are great for listing selected movies to Virtual Directories, but they do not work when you need ABSOLUTE PATHING:

Virtual DIR:  C:\JELLYFIN_HOME\HDD-2\
Actual DIR:  G:\

1st: The Engine that makes it “GO”

Helper: VBScript “Replace” String
Usage:  replace.vbs (INPUT/OUTPUT-FILE) Search-String Replacement-String 

C:\POWER-SHELL-SCRIPTS\Replace.VBS

Const ForReading = 1
Const ForWriting = 2

strFileName = Wscript.Arguments(0)
strOldText = Wscript.Arguments(1)
strNewText = Wscript.Arguments(2)
Set objFSO = CreateObject("Scripting.FileSystemObject")
Set objFile = objFSO.OpenTextFile(strFileName, ForReading)
strText = objFile.ReadAll
objFile.Close
strNewText = Replace(strText, strOldText, strNewText)
Set objFile = objFSO.OpenTextFile(strFileName, ForWriting)
objFile.WriteLine strNewText
objFile.Close

NOTE:  Following Upgraded: ALERT User to Manually Change File Names if they contain an Ampersand (&) with a simple Text Find String Command (Findstr)

5-Drive-Replace-HOME.bat

Echo Checking for Amersands - Breaks Playlist
