Creating YouTube Video w/multiple songs/images for each song section.

The very basic idea is you want to create a video for YouTube where each section of a video with a different song has its own picture.  Given: You have six mp3 songs and six png pictures  each with a corresponding mp3/png name, for example MySong.mp3 and MySong.png. 

FILES needed: 
01_PNG_and_MP3_Spaces_to_Underlines.bat
02_DIR_PNG.bat
03_Call_Join_Video_Audio_MP4.bat
03.5_2BAT_CALL_and_JOIN_and_CONCAT.bat
04_Join_Video_Audio_MP4.bat
05_DIR_MP4.bat
06_CONCAT_MP4.bat

SECTION TITLES Ordered

  • ONE: Download Images (Must be same Dimensions and PNG format) to Working Directory
  • TWO: Download Music (MP3 Format) to working Directory
  • THREE: Create MP4 for each MP3 song
  • FOUR: Concatenate the Video Files
  • FIVE: Create a time index and Upload

SECTION Breakdown


ONE: Download Images using Greenshot or Google Image Search
TWO: Download your Music from Sound Cloud Account (Don’t forget SoundCloud download permissions)
THREE: PREP: 
      NOTE:  PNG and SONG PAIR must have the same name (MySong.mp3 and MySong.png)
      RUN:    01_PNG_and_MP3_Spaces_to_Underlines.bat

@echo off
Setlocal enabledelayedexpansion

Set "Pattern= "
Set "Replace=_"

For %%a in (.png) Do (
    Set "File=%%~a"
    Ren "%%a" "!File:%Pattern%=%Replace%!"
)

For %%a in (.mp3) Do (
    Set "File=%%~a"
    Ren "%%a" "!File:%Pattern%=%Replace%!"
)

      RUN:   02_DIR_PNG.bat to create DirListPNG.txt
@Echo Off
Dir /b *.png > DirListPNG.txt
Example of DirListPNG.txt Format:

01Felicity.png
02The_Riff_Tide.png
03Lazy_Dayz_of_Summer.png
04Reminiscing.png
05Variations_on_a_Theme_by_Vince_Guaraldi.png
06Impermanence.png

EDIT and ADD Lines from DirListPNG.txt to 03_Call_Join_Video_Audio_MP4.bat.txt.

RUN:  03.5_2BAT_CALL_and_JOIN_and_CONCAT to REMOVE .txt safety to activate .bat cmd status.

@Echo Off

ren 03_Call_Join_Video_Audio_MP4.bat.txt 03_Call_Join_Video_Audio_MP4.bat
ren 04_Join_Video_Audio_MP4.bat.txt 04_Join_Video_Audio_MP4.bat
ren 06_CONCAT_MP4.bat.txt 06_CONCAT_MP4.bat

RUN:  03_Call_Join_Video_Audio_MP4.bat
@Echo Off
setlocal enableDelayedExpansion

REM First Argument is PNG file without extension

Set cmdName=04_Join_Video_Audio_MP4.bat

REM  Where MySong and MySong.mp3 have the same name (name.ext)
REM  Call !cmdName! MySong
Here is the Helper BAT file:  04_Join_Video_Audio_MP4
@Echo Off
setlocal enableDelayedExpansion

REM This is a joining of a single PNG to a single MP3 to create a single MP4 before concat. vids
REM Defing Time Variables as Command Line Arguments
set fileName=%1

ffmpeg -r 1 -loop 1 -i !fileName!.png -i !fileName!.mp3 -acodec copy -r 1 -shortest -vf scale=640:360 !fileName!.mp4
Echo --- !fileName! DONE ---
Now you should have an MP4 file for each SONG(MP3)/PICTURE(PNG) Pair.
For Example:

01Felicity.mp4
02The_Riff_Tide.mp4
03Lazy_Dayz_of_Summer.mp4
04Reminiscing.mp4
05Variations_on_a_Theme_by_Vince_Guaraldi.mp4
06Impermanence.mp4

FOUR:

RUN:  05_DIR_MP4.bat
NOTE:  The word “file” is a parameter needed by FFMPEG.
@Echo Off
REM Create File List
for %%i in (*.mp4) do echo file '%%i'>> DirListMP4.txt
Creates DirListMP4.txt

Example of DirListMP4.txt Format:
file '01Felicity.mp4'
file '02The_Riff_Tide.mp4'
file '03Lazy_Dayz_of_Summer.mp4'
file '04Reminiscing.mp4'
file '05Variations_on_a_Theme_by_Vince_Guaraldi.mp4'
file '06Impermanence.mp4'
RUN:  06_CONCAT_MP4.bat

@Echo Off

REM Concatenate Files
ffmpeg -f concat -safe 0 -i DirListMP4.txt -c copy Larryz_Music_This_Month.mp4
You should now have a new file which you can rename to whatever:

Larryz_Music_This_Month.mp4

FIVE:

While Listening to the completed file, take note of the start of each song so you can add it into the comments section of the youtube video so the end user has a way to click forward to the any song in the list.

Here is an example of an Index:

00:03:42 The Riff Tide
