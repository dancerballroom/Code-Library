Creating a Karaoke CD (School of Trial&Err)

Software:

  • Karaoke Builder Studio v3.0.300 (to sync the words with the musci)
  • Power CD+G Burner (to burn the Karaoke CD)
  • Audacity (to change the Sample Rate (44100) and the Bit Rate (16-bit) and save as WAV)
  • PotPlayer ((64-bit) to download MP4 in Windows TEMP directory)
  • FFmpeg (To convert MP4 into MP3)

Hardware:

  • A CD+G Capable Burner:  See notes below at bottom.

My Steps-Prep:

  • Download the backing track for the Karaoke using PotPlayer and YouTube. 
  • Change Name to SONG.mp4 extension.
  • Extract SONG.wav from SONG.mp4 using FFmpeg, or record directly to Audacity.
  • Edit SONG.wav with Audacity.  Resample the Song to 16-bit, Stereo, 41000 bit rate, both on the song and the project bit rate (located in a box at the bottom left).
  • (IMPORTANT NOTE:  If you try to burn your disk onto CD without 16-Bit Stereo 41000 bit rate an error will occur:  Invalid audio format in Project file.  Most of the time this is a problem I have in Audacity not setting the Project Rate in the bottom left box.)
  • (IMPORTANT NOTE:  Resampling does not change the pitch IF you use MENU: Tracks/Resample.)
  • Download the lyrics for the Karaoke.  Save as SONG.txt
  • EDIT SONG.txt Add Spaces to text every 4-6 lines, depends on FONT.
  • Group Versus and Chorus and shorten long lyric lines.  (4-6 max)
  • Add Song and Artist Info at top of text. 

Steps-Karaoke:

  • Open project in Karaoke Builder Studio.
  • Change Font, Color, Text Sweeping
  • Import the SONG.txt
  • Import the SONG.wav
  • Rehearse and then Sync Lyric with Song as close as possible using the space key.
  • Click EDIT ICON to EDIT TAB 1: Lyric Sweeping:  Lyric WORD DURATION
  • EDIT TAB 2: Display/Remove:  Lyric LINE VISUAL DURATION
  • Save and click OK
  • BUILD ICON and create SONG.cdg (the Graphic).  TEST (Note Wave and SONG must have same name)
  • TOOLS ICON to create AVI for YouTube (NOTE: Try each CODEC, I recommend XVID)

Burning the CD, Uploading the AVI, Sharing

  • Power CD+G Burner to open CDG and burn to CD.
  • Upload AVI to YouTube Account
  • Share on Facebook

Note: CDG Compatible Burners

Source:  http://www.karaoke-software.net/karaoke-disc-burners.html

For a burner to be "CDG compatible" it needs to be able to read and write subchannel data. This is information that is written to the usually unused subchannel of a sound track.

In a CDG song track this would be the graphic information in the form of lyrics and images.

Not long ago you would be told that you MUST have a Plextor or some other specific make and model of burner in order to burn a CDG disc. While you still need such a drive to be able to play CDG discs with some computer programs you no longer need one to make copies of CDG discs or to rip song tracks from them.

Modern software applications like Microstudio and Power CD+G Burner work with most modern burners. An easy way to find out if your burner is one of these is to download a trial copy of Power CD+G Burner and see if it works.
