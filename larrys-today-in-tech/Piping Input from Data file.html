2. RUN File

Pipe-Input-Run.bat (The Data Retrieval Batch File)
======
@echo off
Setlocal enabledelayedexpansion

:: Line by Line
:: Input Data and build the command string
:: Handles %1 through %9
for /F "tokens=*" %%A in (Pipe-Input-DATA.txt) do (
    IF "!line!"=="" (
        set line="%%A"
    ) ELSE (
        set line=!line! "%%A"
    )
)

:: Run the Batchfile using Input Data from Pipe-Input-DATA 
Call Process-Input.bat !line!

pause

3. Batch file requesting USER INPUT

Process-Input.bat (The Main Batch File that prompts for USER INPUT)
======
@echo off
Setlocal enabledelayedexpansion

:: %1 through %4 
:: Can Handle %1 through %9
set First_Name=%~1
set Last_Name=%~2
set Profession=%~3
set Age=%~4

:: Prompt if any variables %1 through %4 undefined
:: Prompt if any variables %1 through %4 undefined
if not defined First_Name set /p "First_Name=Candidate First Name: "
if not defined Last_Name set /p  "Last_Name=Candidate Last Name:  "
if not defined Profession set /p "Profession=Candidate Profession: "
if not defined Age set /p        "Age=Candidate Age:      "
Echo.
Echo Processed Entry

:: Print result
echo First Name: %First_Name%
echo Last Name:  %Last_Name%
echo Profession: %Profession%
echo Age:        %Age%
