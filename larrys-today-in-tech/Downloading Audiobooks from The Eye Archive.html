Windows:
  • Install Cygwin (https://cygwin.com/install.html)
    • Include all “bash” packages
    • Include all “wget” packages
  • At the bottom of each page of the eye there is a wget command
      • Don’t use the following, it’s too *.*, Click on a subdirectory and tryj that.
      • wget -m -np -c -U "eye01" -R "index.html*" "https://the-eye.eu/public/AudioBooks/“
    • Click “copy”
    • Click “Copy to Windows clipboard double-folder icon”
  • Open Cygwin
    • Middle mouse wheel-button to paste wget command
    • Press enter
    • NOTE:  At some point The download will either try again if not working and download, or it will fail due to Network congestion.  
    • If it stalls out, focus should be on the Cygwin window when you press “Control + C” to stop the download attempt and try again.
    • Press “Up-Arrow” to bring up the last command. (doskey like cmd history for the bash shell)
    • Press Enter to restart the download command.  anything already downloaded will be skipped.  anything partially downloaded will be continued and completed.
    • NOTE:  You can use windows to download each file, but as Zachary Smith would say on Lost in space:  “Oh, the pain.  the pain.  the pain.”
