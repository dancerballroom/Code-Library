CCCC Chop Chop S05 E8.0.bat Example 

Copy and Change 60+ times, One for Each Second in a Minute and above

Just change the Yellow Highlighted Numbers for Seconds to be Cut

Note:  Syntax Highlighting Issues are QUIP issues ONLY.
Echo Off
setlocal enableDelayedExpansion
COLOR 0A
TITLE Chop off the first 5 seconds and last 8.0 seconds for all MP4 files in CurDir

Echo Remove from the start of *.MP4: 5.0 seconds and end of *.MP4: 8.0 seconds
Echo Place in the Trimmed Directory

echo Time Started >> Trimmed_Processed_List.txt 
time /t >> Trimmed_Processed_List.txt 

Set "ext=mp4"
Set "opts=-v quiet"
Set "opts=%opts% -print_format "compact=print_section=0:nokey=1:escape=csv""
Set "opts=%opts% -show_entries "format=duration""
If Exist *.%ext% (If Not Exist "Trimmed\" MD Trimmed)
For %%a In (*.%ext%) Do Call :Sub "%%~a"
Exit /B

:Sub
echo passed SUB
echo %1
echo entering FOR with FFProbe
echo opts = %opts% 
For /f "Tokens=1* Delims=." %%a In ('FFProbe %opts% %1') Do (Set /A "ws=%%a-8.0" & Set "ps=%%b")
rem If %ws% Lss 20 GoTo :EOF
echo passed 1st FOR
Set/A hh=ws/(60*60), lo=ws%%(60*60), mm=lo/60, ss=lo%%60
echo passed SET
If %hh% Lss 10 Set hh=0%hh%
If %mm% Lss 10 Set mm=0%mm%
If %ss% Lss 10 Set ss=0%ss%

echo right before FFMPEG command
FFMpeg -itsoffset -5.0 -i %1 "Trimmed\%~1.srt"
FFMpeg -i %1 -ss 00:00:05 -to %hh%:%mm%:%ss% -c:v copy -c:a copy "Trimmed\%~1"

echo ***************************************************
echo ****************** DONE ***********************
echo ***************************************************

echo ------------------------------ >> Trimmed_Processed_List.txt
echo DONE: File, Time Complete, and File Length >> Trimmed_Processed_List.txt 
echo Trimmed\%~1 >> Trimmed_Processed_List.txt 
time /t >> Trimmed_Processed_List.txt 
echo %hh%:%mm%:%ss% >> Trimmed_Processed_List.txt 
echo ------------------------------ >> Trimmed_Processed_List.txt
