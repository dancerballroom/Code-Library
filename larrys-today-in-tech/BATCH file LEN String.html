BATCH file LEN String

Example:  Test-Len-String.bat

@Echo Off

SET strvar=My Bonnie Lies over the ocean
REM This is Len 29

ECHO %strvar%> tempfile.txt
FOR %%? IN (tempfile.txt) DO ( SET /A strlength=%%~z? - 2 )

Echo The length of the string is %strlength%

Output

C:\Users\Larry\Desktop\Uses-String-Replace\EXPY2>Test-Len-String.bat
The length of the string is 29
Press any key to continue . .

NOTE:

You can do it in two lines, fully in a batch file, 
by writing the string to a file and then getting the 
length of the file. You just have to subtract two bytes 
to account for the automatic CR+LF added to the end.
