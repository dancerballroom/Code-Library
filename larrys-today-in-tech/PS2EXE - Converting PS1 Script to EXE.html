PS2EXE - Converting PS1 Script to EXE

1.  Install the PS2EXE Module into (PowerShell: Choose Run As Admin)

PS C:\WINDOWS\system32>Install-Module -Name ps2exe -RequiredVersion 1.0.5

# PS2EXE is now Installed.

Here is a simple run of the test PowerScript File (Sleep.ps1)

Example Usage:  Sleep 7 (from the PowerShell Environment or the PS1 Editor)
Result: The Process goes to sleep for 7 seconds.

Sleep.ps1 (PowerShell Script)

# Sleep for n Seconds
$Seconds=$args[0]
Start-Sleep $Seconds

2.  Create the Executable (EXE) 

Syntax:  PS2EXE <INPUT.PS1> <OUTPUT.EXE>

PS C:\POWER-SHELL-SCRIPTS> ps2exe Sleep.ps1 Sleeper.exe
PS2EXE-GUI v0.5.0.21 by Ingo Karstein, reworked and GUI 
support by Markus Scholtes

3.  Test The EXE

From PowerShell Environment:

PS C:\POWER-SHELL-SCRIPTS> .\Sleeper 7
