Burn-In-Subs.bat

Name: Burn-In-Subs.bat

Goal: Convert Movies on-the-fly  that have Subtitles.srt files.  This is mainly for Streaming Subtitles to equipment that do not keep up with Subtitle SRT files integration creating pesky latency issues.

Purpose: Automatically Create a MP4 Movie or Show with the Subtitles burned in using Windows Send-To Directory.  

Output:  Creates a Subdirectory called “Burned-In-Subs“ and places the movie copy with burned in subtitles there.

Note:  After Installed the Batchfile Script can be accessed from any folder on the computer. 

Works:  06/25/2022

The Burn-In-Subs.bat Script:

@Echo Off
setlocal enabledelayedexpansion
REM ***** The Basics *****
REM NAME: Burn-In-Subs.bat
REM LOCATION of SCRIPT: Windows SendTo Directory
REM       To Operate this Script:
REM       For Every Movie.mp4 there must be a Movie.srt
REM             "The Green Mile.mp4" and "The Green Mile.srt"
REM             "Star Wars.mp4" and "Star Wars.srt"
REM             etc.
REM ***** To INSTALL *****
REM     Download FFMPEG and place ffmpeg.exe in C:\FFMPEG\bin
REM     Place this Batchfile in the Windows Send-To DIRECTORY
REM       To show the SendTo DIR:  Windows+R  THEN:  shell:sendto
REM ***** INSTRUCTIONS *****
REM     Right Click on SomeVideo.MP4
REM     Hover over SendTo
REM     Click on "Burn-In-Subs.bat"
REM     (Takes a minute or two and DONE!)
REM ***** Other info and Requirements *****
REM The Command is Based ON the following Code (from the FFMPEG website)
REM (NOTE: The following is changed to 2 LINES for visibility in QUIP)
REM ffmpeg -i "The Incredibles.mp4" -vf subtitles="The Incredibles.srt" 
REM        -c:a copy "Burned-In-Subs\The Incredibles.mp4"
REM
REM ***** The Givens *****
REM     FFMpeg.exe is located in C:\FFMPEG\bin   
REM           To Change Location--> SET the FFMPEG_DIR Variable
REM     The SRT file has the same name as MOVIE  
REM           If Incroyable.srt--> Change to Incroyable.srt on MAIN command
REM     The SRT is in the format MOVIE.SRT       
REM           If JAWS.ENG.srt--> Change MOVIE.ENG.srt on MAIN command
REM     Later to Improve:  There are no special characters in the filename 
REM           including Commas, Apostrophes, or Colons allowed at this time.

REM ***** SET Variables *****
SET "FFMPEG_DIR=C:\FFMPEG\bin"
SET "FFMPEG_CMD=ffmpeg.exe"
Set "DI=%~d1"
SET "PI=%~p1"
SET "NI=%~n1"
SET "MOVIE=%NI%"
SET "MOV_DIR=%DI%%PI%"

cls
pushd %MOV_DIR%

If Not Exist "Burned-In-Subs\" MD Burned-In-Subs
REM "  <-- Fix Syntax Highlighting in QUIP
REM The Command that makes everything work.
%FFMPEG_DIR%\%FFMPEG_CMD% -i "%MOVIE%.mp4" -vf subtitles="%MOVIE%.srt" -c:a copy "Burned-In-Subs\%MOVIE%.MP4"

popd
echo Created:  Burned-In-Subs\%MOVIE%.MP4
Pause

END of Batch Script
